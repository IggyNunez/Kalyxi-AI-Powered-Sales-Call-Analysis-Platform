Setup Instructions

Follow these steps to configure the extension:

Connect Google Account

Go to Kalyxi Dashboard and connect your Google account.

Create an Extension Token

In the dashboard, create a new extension API token and copy it.

Paste Token Below

Enter your API token in the field below and click Save.

API Configuration

Enter your Kalyxi API credentials

Get your token from the Google Connections page.

Only change this if using a self-hosted instance.
